position static

This is the default position of all HTML elements. The element is placed according to the normal document flow. It does not respond to properties like top, left, right, or bottom.

position relative

The element remains in the normal document flow but can be adjusted using top, left, right, or bottom. It moves relative to its original position without affecting other elements.
